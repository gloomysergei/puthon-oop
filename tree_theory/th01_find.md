
### Mетод __find

`s, p, fl_find = self.__find(self.root, None, obj.data)`

В результате вызова метода `__find()` возвращаются три значения, которые сразу присваиваются трём переменным:

- `s (node)` — найденный узел или узел, рядом с которым должен располагаться искомый элемент:
  - если элемент найден — это узел с искомым значением;
  - если не найден — это «последний» узел на пути поиска (тот, у которого отсутствует нужный потомок).

- `p (parent)` — родитель узла `s`:
  - нужен для операций удаления и вставки;
  - может быть `None` (если `s` — корень дерева).

- `fl_find (found)` — флаг наличия элемента `(True/False)`:
  - `True` — элемент найден в дереве;
  - `False` — элемент не найден.

### Как работает метод `__find()`

Метод выполняет поиск в бинарном дереве поиска по алгоритму:

1. Начинает с корневого узла.

2. Сравнивает искомое значение с текущим узлом:

    - если равны — возвращает текущий узел, его родителя и True;
    - если искомое меньше — переходит к левому потомку;
    - если искомое больше — переходит к правому потомку.

3. Повторяет шаги до:

    - нахождения элемента;
    - достижения None (конца ветки).


### Пример кода

```python

class Node:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

class Tree:
    def __init__(self):
        self.root = None
    
    def __find(self, node, parent, value):

        # базовый случай рекурсии. Дошли до конца ветки дерева
        if node is None:
            return None, parent, False
        
        if value == node.data:
            return node, parent, True
        
        if value < node.data:
            if node.left:
                return self.__find(node.left, node, value)
            else:
                return node, parent, False
        else:  # value > node.data
            if node.right:
                return self.__find(node.right, node, value)
            else:
                return node, parent, False
    
    def append(self, obj):
        if self.root is None:
            self.root = obj
            return obj
        
        # Вот та самая строка!
        s, p, fl_find = self.__find(self.root, None, obj.data)
        
        if not fl_find and s:
            if obj.data < s.data:
                s.left = obj
            else:
                s.right = obj
        return obj
```

### Практические примеры использования

#### Пример 1. Вставка элемента

Допустим, есть дерево:

    10
   /  \
  5    15

Нужно вставить 7.
*Вызов:*

```python
s, p, fl_find = self.__find(self.root, None, 7)
```

*Результат:*

- s = узел с 5 (последний узел на пути);
- p = узел с 10 (родитель s);
- fl_find = False (7 не найден).

*Далее*
 код проверяет: `if not fl_find and s: → True`, и добавляет 7 как правый потомок узла 5.

Пример 2. Поиск существующего элемента

Ищем 5 в том же дереве:

    s = узел с 5;

    p = узел с 10;

    fl_find = True.

Типичные сценарии использования

    Вставка (append):

        найти место для нового узла;

        прикрепить его к соответствующему потомку узла s.

    Удаление (del_node):

        найти удаляемый узел (s);

        получить его родителя (p) для обновления связей;

        проверить существование (fl_find) — если False, удалять нечего.

    Поиск:

        использовать fl_find для ответа «есть/нет»;

        использовать s для доступа к данным узла.

Краткий итог

Запись s, p, fl_find = self.__find(...) — паттерн для работы с BST, который:

    находит узел или место для узла (s);

    даёт доступ к родителю для модификации дерева (p);

    предоставляет флаг успеха операции (fl_find).

Этот подход позволяет выполнять все базовые операции (поиск, вставка, удаление) с минимальным числом обходов дерева — всё нужное получается за один вызов __find()
