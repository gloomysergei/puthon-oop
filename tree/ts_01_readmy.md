## Общая структура

Код состоит из двух функций:

1. Внешняя функция `solution(table_of_contents)` — точка входа, принимает данные оглавления.
2. Вложенная рекурсивная функция `walk(table_of_contents, indent='')` — выполняет основную работу по обходу структуры.

### Детальный разбор

**Функция `solution`**

```python
def solution(table_of_contents):
    return '\n'.join(walk(table_of_contents))
```

Делает следующее:

- принимает структуру оглавления `(table_of_contents)`;
- вызывает вспомогательную функцию `walk()` для получения списка строк;
- объединяет элементы списка в единую строку, разделяя их символами переноса строки `(\n)`.

**Вложенная функция `walk`**

```python
def walk(table_of_contents, indent=''): #  indent ['ɪndent] - отступ
    chapters = []
    for chapter in table_of_contents:
        chapters.append(f"{indent}{chapter['title']}")
        if 'chapters' in chapter:
            chapters.extend(walk(chapter['chapters'], indent=indent + '* '))
    return chapters
```

**Параметры:**

- `table_of_contents` — текущий уровень структуры оглавления (список словарей);
- `indent` — строка отступа для текущего уровня вложенности (по умолчанию пустая).

**Пошаговая логика:**

1. Создаёт пустой список chapters для хранения строк оглавления на текущем уровне.
2. Перебирает все элементы `chapter` в текущей структуре оглавления:

    - добавляет заголовок главы в список с текущим отступом: `chapters.append(f"{indent}{chapter['title']}")`;
    - проверяет, есть ли вложенные главы: `if 'chapters' in chapter`;
    - если вложенные главы есть, рекурсивно вызывает себя для обработки вложенного уровня, увеличивая отступ на `'* '`: `chapters.extend(walk(chapter['chapters'], indent=indent + '* '))`.
    - Возвращает собранный список строк для текущего уровня.

**Как работает рекурсия:**

Функция `walk` вызывает сама себя для каждого уровня вложенности:

- на верхнем уровне отступ пустой `('')`
- для первого уровня вложенности отступ становится `'* '`
- для второго уровня — `'* * '` и т. д.

Каждый рекурсивный вызов работает со своим списком `chapters`, а результат объединяется через `extend()`.

`extend()` — встроенный метод списков в `Python`, который добавляет элементы из итерируемого объекта в конец существующего списка. В отличие от `append()`, он не добавляет объект целиком, а «распаковывает» его и добавляет каждый элемент отдельно.
